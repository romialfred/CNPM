# Accessibility rules

- Target WCAG 2.2 AA.
- No critical/serious axe issue.
- Keyboard and visible focus for all actions.
- Correct landmarks, headings and page title updates.
- Labels and errors programmatically associated.
- Status not conveyed by color alone.
- Dialog focus trapped and restored.
- Reflow at 320 CSS px and zoom 200%.
- Chart has summary/table alternative.
- Reduced motion respected.
