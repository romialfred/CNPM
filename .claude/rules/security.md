# Sécurité
- Principe du moindre privilège et refus par défaut.
- 2FA obligatoire pour les profils sensibles et step-up pour les opérations critiques.
- Interdire les secrets dans le dépôt, les logs, les captures et les messages d’erreur.
- Valider type, taille, extension, signature et antivirus pour tout fichier reçu.
- Les exports sensibles sont approuvés, chiffrés, expirables et audités.
- Toute vulnérabilité critique bloque la livraison.
