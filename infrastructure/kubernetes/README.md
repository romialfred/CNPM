# Kubernetes / RKE2

Les manifestes de production sont volontairement non figés tant que les décisions d’hébergement, de capacité et de SLA ne sont pas approuvées. La cible doit inclure : namespaces par environnement, NetworkPolicies, PodDisruptionBudgets, ressources, probes, autoscaling contrôlé, secrets externes, stockage chiffré, ingress/WAF, sauvegarde et observabilité.
