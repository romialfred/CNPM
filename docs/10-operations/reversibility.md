# Réversibilité

Le CNPM peut obtenir à tout moment :
- un dump PostgreSQL documenté et restaurable ;
- des exports CSV/JSON structurés par domaine ;
- les documents dans leur format original avec manifestes et empreintes ;
- les journaux d’audit et configurations non secrètes ;
- le code source, les migrations, contrats, runbooks et inventaire de dépendances ;
- la procédure de reconstruction des environnements.

Un test de réversibilité est réalisé avant le passage en Phase 2 puis annuellement.
