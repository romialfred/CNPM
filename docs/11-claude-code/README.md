# Mode opératoire des agents de développement

Les instructions transversales sont à la racine dans `CLAUDE.md` et `AGENTS.md`. Les règles exécutables et spécialisées sont sous `.claude/`.

Ce dossier conserve les documents de travail complémentaires :

- `operating-model.md` — organisation des sessions et des revues ;
- `prompt-library.md` — modèles de demandes contrôlées.

Ne pas dupliquer ici les règles déjà présentes dans `.claude/rules/` ou `.claude/skills/`.
