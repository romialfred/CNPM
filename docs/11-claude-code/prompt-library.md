# Bibliothèque de prompts

## Implémentation
`Implémente US-PAY-003 en utilisant la skill implement-user-story. Ne modifie pas le contrat OpenAPI ou le modèle PostgreSQL sans présenter d’abord le diff de conception.`

## Revue
`Utilise les agents database-reviewer, api-contract-reviewer et security-reviewer sur cette pull request. Classe les constats et cite les fichiers/lignes.`

## Migration
`Crée la prochaine migration Flyway pour cette story avec la skill create-flyway-migration. Inclure contraintes, index, test N-1 et stratégie de retour arrière.`

## Release
`Évalue la release avec la skill release-readiness. Produis une décision Go, Go conditionnel ou No-Go avec preuves manquantes.`
