# Politique des données de test

- Données synthétiques par défaut.
- Aucun NIF, RCCM, téléphone, courriel ou paiement réel en environnement non productif.
- Jeux reproductibles, versionnés et réinitialisables.
- Secrets et jetons factices clairement identifiés.
- Les données de performance peuvent être générées à grande échelle sans reproduire des personnes réelles.
