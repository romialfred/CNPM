# Classification des données

| Niveau | Exemples | Contrôles minimaux |
|---|---|---|
| Public | Informations institutionnelles publiées | Intégrité et validation éditoriale |
| Interne | Paramètres non sensibles, statistiques agrégées | Authentification et journalisation |
| Confidentiel | Fiches membres, contacts, cotisations | Moindre privilège, chiffrement, masquage |
| Restreint | Identifiants, pièces KYC, exports nominatifs | 2FA, approbation, chiffrement renforcé, accès audité |
| Secret | Clés, jetons, secrets partenaires, charges bancaires complètes | Coffre-fort, rotation, accès exceptionnel, aucune journalisation |

La durée de conservation et la base juridique doivent être validées par le CNPM et son conseil juridique avant la production.
