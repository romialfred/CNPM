# Actifs de marque CNPM

Le seul actif de marque canonique actuellement fourni est :

```text
docs/00-sources/logo-CNPM.png
```

Aucune reconstruction vectorielle, variante de couleur, favicon ou logo généré n’est approuvé pour la production. Les écrans de référence contiennent le logo uniquement comme élément de contexte visuel.

Lorsqu’un SVG institutionnel validé sera remis par le CNPM, le déposer dans `docs/00-sources/` et mettre à jour la décision de marque dans `docs/00-governance/open-decisions.md`.
