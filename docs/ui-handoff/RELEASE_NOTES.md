# Notes de livraison — Handoff UI/UX intégré v1.0.0

## Contenu

- 101 écrans inventoriés ;
- 74 composants catalogués ;
- tokens Web, TypeScript, Flutter et ECharts ;
- 14 références visuelles et leurs fiches ;
- règles responsive, accessibilité WCAG 2.2 AA et régression visuelle ;
- site public, portail membre, back-office, mobile et vitrine publique des membres ;
- fixtures, schémas JSON, templates, guide Word/PDF et matrice opérationnelle.

## Intégration au dépôt

L’overlay historique a été fusionné dans `.claude/`, `web/` et `mobile/`. Les sources institutionnelles dupliquées et variantes de logo non officielles ont été supprimées. Le handoff est désormais conservé sous `docs/ui-handoff/`.

## Décisions ouvertes

Consulter `docs/00-governance/open-decisions.md` à la racine et `data/open-decisions.json` pour les décisions UI détaillées.
