# Iconographie

## Bibliothèque proposée

Utiliser **Lucide** comme bibliothèque unique de pictogrammes d’interface, sous réserve de validation finale. Les logos, badges officiels et symboles de paiement sont des actifs distincts.

## Tailles

- 14 px : pictogramme d’accompagnement dans une table dense.
- 16 px : bouton compact et métadonnée.
- 20 px : contrôle standard.
- 24 px : navigation et actions principales.
- 32 px : illustration fonctionnelle d’un état vide.

## Règles

- Un bouton uniquement iconique possède un nom accessible et, sur desktop, un tooltip.
- L’épaisseur visuelle des icônes doit rester cohérente.
- Ne pas mélanger Lucide, Material, Font Awesome et SVG hétérogènes dans un même produit.
- Ne jamais utiliser une icône ambiguë sans libellé pour une action financière.
- Les icônes ne remplacent pas les libellés de statut.
