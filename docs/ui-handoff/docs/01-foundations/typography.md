# Typographie

## Famille

Police recommandée : **Inter**, sous réserve de validation et de licence. La pile de secours est :

```css
Inter, ui-sans-serif, system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif
```

Aucun fichier de police n’est livré dans ce pack.

## Échelle

| Style | Taille / ligne | Graisse | Usage |
|---|---:|---:|---|
| Display | 48 / 58 px | 700 | Hero public uniquement |
| H1 | 36 / 44 px | 700 | Titres publics majeurs |
| H2 | 30 / 38 px | 700 | Sections publiques |
| Page title | 24 / 32 px | 700 | Back-office et portail membre |
| Section title | 18 / 28 px | 600 | Cartes et sections |
| Body | 16 / 24 px | 400 | Contenu public et formulaires |
| Body small | 14 / 20 px | 400 | Tables et aides |
| Caption | 12 / 16 px | 400/500 | Métadonnées |
| Metric | 24–32 / 32–40 px | 700 | KPI |

## Règles

- Une page contient un seul `h1`.
- Les titres de cartes ne sautent pas de niveau arbitrairement.
- Le texte entièrement en capitales est réservé aux micro-libellés de marque ou catégories très courtes.
- La largeur de ligne éditoriale cible 65 à 75 caractères.
- Les montants utilisent des chiffres tabulaires lorsque la police le permet.
- Les labels de champs sont visibles en permanence ; le placeholder n’est jamais le seul label.
- Les tailles inférieures à 12 px sont interdites.
