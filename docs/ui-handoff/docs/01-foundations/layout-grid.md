# Grille et mise en page

## Desktop de référence

- Viewport de référence : **1672 × 941 px**.
- Sidebar étendue : **252 px**.
- Sidebar réduite : **80 px**.
- Topbar : **72 px**.
- Padding de page : **28 px**.
- Grille : **12 colonnes**, gouttière **24 px**.
- Espace vertical entre sections : **24 px**.
- Padding standard des cartes : **20 px**.

## Largeurs de contenu

- Back-office : fluide, maximum 1920 px ; la table peut occuper la largeur disponible.
- Site public : conteneur maximum 1440 px, avec sections full-width contrôlées.
- Formulaire : largeur utile recommandée 880–1040 px + panneau d’aide 280–320 px.
- Texte éditorial : 720–780 px maximum.

## Alignements

- Les titres, contrôles de filtre et tableaux partagent le même axe gauche.
- Les nombres d’une colonne financière sont alignés à droite.
- Les actions d’en-tête sont alignées sur le titre, puis passent à la ligne de façon prévisible.
- Une carte ne doit pas être utilisée comme simple espaceur.

## Scroll

- La page gère le scroll principal ; éviter les multiples zones verticales imbriquées.
- Les tables larges peuvent défiler horizontalement sur tablette, avec première colonne épinglée si nécessaire.
- Les barres d’action sticky ne masquent jamais le contenu ni le focus.
