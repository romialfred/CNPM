# Règles de marque CNPM

## Source actuelle
Le seul logo fourni et conservé dans le dépôt est `docs/00-sources/logo-CNPM.png`. Sa définition et son canal alpha ne permettent pas de le considérer comme un actif de production définitif.

Aucune reconstruction, variante de handoff ou série de favicons non validée n’est incluse dans le dépôt consolidé.

## Exigence avant production
Le CNPM doit fournir :
- un SVG ou fichier vectoriel officiel ;
- les références de couleurs certifiées ;
- les variantes sur fond clair, sombre et monochrome ;
- la zone de protection et la taille minimale ;
- les droits et règles d’utilisation ;
- les icônes et favicons officiellement approuvés.

## Usage temporaire
Pendant le développement, utiliser le raster source comme placeholder sans le redessiner, le déformer, le recolorer ni l’intégrer dans un actif présenté comme officiel. Les captures conceptuelles restent des références de composition et non des sources de marque.
