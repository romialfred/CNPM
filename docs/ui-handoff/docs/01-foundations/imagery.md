# Photographie et médias

## Site public et vitrines membres

Les images doivent montrer des entreprises, équipes, infrastructures, services et activités réelles au Mali ou dans leur contexte pertinent. Éviter les banques d’images génériques sans rapport avec le membre.

## Formats recommandés

| Usage | Ratio | Taille minimale |
|---|---:|---:|
| Hero public | 16:9 ou 3:2 | 1600 × 900 px |
| Carte projet | 4:3 | 800 × 600 px |
| Galerie | 4:3 / 1:1 | 1200 px côté long |
| Logo membre | variable, fond transparent | 512 px côté long |
| Portrait contact | 1:1 | 512 × 512 px |
| Brochure | PDF accessible | selon document |

## Traitement

- Recadrage focal configuré par l’éditeur.
- Compression WebP/AVIF avec fallback selon la cible.
- `srcset` et tailles responsives.
- Texte alternatif obligatoire pour une image informative.
- Alt vide pour les images purement décoratives.
- Pas de texte essentiel incrusté dans l’image.
- Pas de logos partenaires sans autorisation.

## Modération

Chaque média chargé comporte propriétaire, source, consentement, texte alternatif, date de publication et statut de modération.
