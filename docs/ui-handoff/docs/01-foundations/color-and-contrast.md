# Couleurs et contraste

## Règles d’usage

- Texte principal sur blanc : `#101828`.
- Texte secondaire sur blanc : `#475467`.
- Bleu CNPM sur blanc : adapté aux titres et actions.
- Rouge CNPM sur blanc : CTA/alerte, pas texte courant.
- Les fonds sémantiques 50/100 accompagnent un texte sémantique 700.
- Un statut comprend toujours un libellé et, si utile, une icône.

## Seuils

- Texte normal : contraste minimum 4,5:1.
- Grand texte : 3:1.
- Composants, focus, graphiques essentiels : 3:1 par rapport aux couleurs adjacentes.
- Focus visible : anneau d’au moins 2 px ou équivalent perceptible.

## Graphiques

- Limiter à huit couleurs catégorielles.
- Les séries sont distinguées par libellé, ordre, forme ou motif, pas uniquement par couleur.
- Les données tabulaires ou un résumé textuel doivent être disponibles.
- Le rouge n’est pas une série neutre : il signale une comparaison critique ou une anomalie.
