# Pattern — Recherche, filtres et tri

## Recherche

- Recherche globale dans la topbar : membres, entreprises, reçus et actions autorisées.
- Recherche de page : limitée à l’objet courant.
- Délai de 250–350 ms pour recherche dynamique.
- Afficher un bouton d’effacement et l’état de chargement.
- Les résultats sont navigables au clavier.

## Filtres

- Les filtres principaux sont visibles ; les filtres avancés peuvent être repliés.
- Les filtres actifs sont représentés par des chips supprimables.
- L’URL conserve filtres, tri, page et onglet lorsque la vue doit être partageable.
- « Réinitialiser » n’est visible que si un filtre est actif.
- Afficher le nombre de résultats après application.

## Tri

- Un seul tri par défaut ; multicolonne uniquement si besoin explicite.
- Le sens est visible et annoncé.
- Les montants et dates sont triés sur leur valeur, pas leur texte formaté.

## Aucun résultat

Expliquer que les filtres ne produisent aucun résultat et proposer de les effacer, sans afficher l’état vide de première utilisation.
