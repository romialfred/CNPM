# Pattern — Notifications et audit

## Notifications

- Centre de notifications avec catégories : paiement, cotisation, requête, document, sécurité, vitrine.
- Statut lu/non lu et préférence par canal.
- Les notifications critiques ne dépendent pas uniquement d’un toast.
- Un clic ouvre l’objet exact et respecte l’habilitation actuelle.

## Audit visible

Les écrans sensibles affichent une piste compréhensible : acteur, action, date, objet, ancien/nouvel état et justification. Les détails techniques restent dans les journaux administratifs.

## Horodatage

Afficher l’heure locale et rendre la date exacte accessible derrière les libellés relatifs. Dans les captures visuelles, utiliser des dates fixes plutôt que « il y a 5 minutes ».
