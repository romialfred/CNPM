# Couverture Storybook

## Composants P0

Chaque composant P0 comporte :

- Default ;
- variantes ;
- tailles ;
- états interactifs ;
- disabled/read-only ;
- loading ;
- erreur ;
- contenu long ;
- locale française ;
- viewport mobile si concerné ;
- test d’interaction ;
- axe ;
- snapshot visuel.

## Pages

Les pages de référence ont une story ou un environnement de test dédié avec réseau mocké. Les données sont déterministes. Les pages financières couvrent au moins le statut nominal, partiel, erreur et conflit.
