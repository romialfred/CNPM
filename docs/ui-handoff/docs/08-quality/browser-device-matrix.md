# Matrice navigateurs et appareils

## Web

| Cible | Niveau |
|---|---|
| Chrome stable et précédente | Principal |
| Edge stable et précédente | Principal |
| Firefox stable et précédente | Supporté |
| Safari macOS stable et précédente | Supporté |
| Safari iOS versions supportées par le projet | Principal portail/public |
| Chrome Android versions supportées | Principal portail/public |

## Résolutions

Voir `data/viewports.json`. Les viewports de référence sont des dimensions CSS, non des dimensions physiques.

## Mobile natif

- Android : au moins un appareil faible/moyen et un appareil récent.
- iOS : petit écran supporté et appareil récent.
- Tester taille de texte élevée, contraste, rotation autorisée selon écran, mode sombre système même si l’app reste claire, et réseau lent/intermittent.

## Impression

Chrome/Edge pour reçus et états imprimables ; la preuve officielle reste le PDF généré.
