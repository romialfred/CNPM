# Workflow Codex pour les interfaces

1. Lire `AGENTS.md` à la racine.
2. Utiliser `docs/ui-handoff/` comme source normative UI.
3. Proposer un plan, implémenter par écran ou composant, puis exécuter les tests.
4. Fournir captures, rapport d’écart, résultats axe et risques résiduels.
5. Ne jamais modifier une baseline pour masquer un défaut.

L’overlay historique a été fusionné dans le dépôt et ne doit pas être recopié.
