# Checklist de livraison du pack dans le dépôt

- [ ] `AGENTS.md` ou `CLAUDE.md` installé.
- [ ] Règles `.claude` installées si Claude Code.
- [ ] Tokens importés et script de génération exécutable.
- [ ] Logo remplacé par SVG officiel si disponible.
- [ ] Police approuvée installée, sans copier des fichiers non licenciés.
- [ ] Icônes standardisées.
- [ ] Storybook fonctionnel.
- [ ] Playwright et axe fonctionnels.
- [ ] Fixtures intégrées dans mocks.
- [ ] Références visuelles accessibles au CI sans données confidentielles.
- [ ] Écrans pilotes validés.
- [ ] Baselines approuvées.
- [ ] Journal des décisions initialisé.
- [ ] Owners Produit, UX, Frontend, Mobile, QA et Accessibilité identifiés.
