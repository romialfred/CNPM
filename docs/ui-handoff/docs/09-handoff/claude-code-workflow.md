# Workflow Claude Code pour les interfaces

1. Lire `CLAUDE.md` à la racine et les règles spécialisées sous `.claude/rules/`.
2. Identifier la story, l’écran et la fiche de référence.
3. Lire les tokens, le catalogue de composants et les patterns applicables.
4. Signaler les ambiguïtés avant de coder.
5. Implémenter avec fixtures déterministes et composants partagés.
6. Exécuter lint, tests, axe et captures aux viewports obligatoires.
7. Comparer à la référence, corriger la structure puis les détails.
8. Documenter les preuves et toute décision nouvelle.

L’overlay historique a été fusionné dans le dépôt et ne doit pas être recopié.
