# Périmètre du handoff UI/UX

## Inclus

- Site institutionnel public du CNPM.
- Annuaire public et fiches publiques des membres.
- Vitrine promotionnelle paramétrable par membre.
- Authentification, récupération de compte et 2FA.
- Back-office complet.
- Portail membre Web.
- Application mobile membre.
- États de chargement, vide, erreur, accès refusé, hors connexion et synchronisation.
- Responsive, accessibilité, contenu, visualisation de données et tests visuels.
- Overlay de dépôt pour Claude Code et Codex.

## Hors périmètre de conception visuelle finale

- Charte éditoriale institutionnelle exhaustive.
- Photothèque définitive.
- Logo vectoriel certifié et manuel de marque officiel.
- Contenu réel de chaque entreprise membre.
- Pages juridiques finales.
- Choix contractuel des fournisseurs de cartes, paiement, SMS ou analytics.
- Égalité de rendu absolue entre tous les moteurs graphiques.

## Niveau de maturité

Le pack est une **baseline prête à implémenter**. Les décisions encore ouvertes sont listées dans `open-decisions.md`. Elles ne bloquent pas la construction des fondations, mais doivent être closes avant la mise en production des fonctions concernées.
