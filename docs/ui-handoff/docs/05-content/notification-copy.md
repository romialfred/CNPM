# Modèles de notifications

## Cotisation à échéance

**Objet :** Votre cotisation CNPM arrive à échéance le {{date}}  
**Message :** Le montant de {{amount}} pour {{period}} est à régler avant le {{date}}. Consultez le détail ou payez depuis votre espace membre.

## Paiement reçu

**Objet :** Paiement reçu — {{reference}}  
**Message :** Votre paiement de {{amount}} a été reçu le {{date}}. Il est en cours de vérification. Le reçu officiel sera disponible après confirmation.

## Reçu disponible

**Objet :** Votre reçu CNPM est disponible  
**Message :** Le reçu {{receiptNumber}} correspondant à {{period}} est disponible dans votre espace membre.

## Requête mise à jour

**Objet :** Mise à jour de votre requête {{requestNumber}}  
**Message :** Le statut est désormais « {{status}} ». Consultez le message du CNPM dans votre espace.

## Vitrine à corriger

**Objet :** Modifications demandées pour votre vitrine publique  
**Message :** La version soumise nécessite des ajustements. Consultez les remarques, modifiez le brouillon puis soumettez une nouvelle version.

Les SMS sont plus courts et contiennent un lien court officiel uniquement. Ne jamais envoyer de secret ou de document sensible par SMS.
