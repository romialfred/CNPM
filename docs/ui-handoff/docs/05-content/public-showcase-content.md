# Guide de contenu — Vitrines membres

## Champs éditoriaux

| Champ | Limite recommandée | Règle |
|---|---:|---|
| Nom | 120 caractères | Nom officiel |
| Tagline | 80 caractères | Proposition claire, sans superlatif invérifiable |
| Résumé | 400–600 caractères | Qui, quoi, pour qui, où |
| Activité | 60 caractères | Nom court |
| Description activité | 180 caractères | Bénéfice/portée |
| Projet | 120 caractères | Titre réel |
| Résumé projet | 300 caractères | Contexte, résultat, date |
| Métatitre SEO | 55–60 caractères | Inclure le nom |
| Métadescription | 145–160 caractères | Descriptive, non trompeuse |
| Alt image | 150 caractères | Décrire le contenu utile |

## Contenus interdits ou à modérer

- promesse trompeuse ou non démontrée ;
- donnée personnelle sans consentement ;
- contenu politique partisan au nom du CNPM ;
- contenu discriminatoire, illégal ou portant atteinte à un tiers ;
- logo ou photo sans droit ;
- coordonnées financières ou secrets ;
- badge, certification ou partenaire non vérifié ;
- document confidentiel.

## Responsabilité

La page indique que le contenu de présentation est fourni par le membre. Le badge CNPM ne valide que les éléments définis par sa politique.
