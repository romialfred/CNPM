# Lecteurs d’écran

## Noms et descriptions

Chaque contrôle possède un nom accessible stable. Les descriptions et erreurs sont reliées avec `aria-describedby`. Les icônes décoratives sont masquées.

## Annonces

- Résultat de recherche : annonce polie du nombre de résultats.
- Sauvegarde automatique : annonce seulement lors d’un changement significatif, sans spam.
- Erreur de formulaire : résumé focalisé.
- Toast : région live appropriée.
- Traitement long : `aria-busy`, progression et fin.
- Navigation SPA : annoncer le nouveau titre de page et positionner le focus au début du contenu.

## Documents et graphiques

Les reçus ont un résumé textuel et un lien PDF. Les graphiques ont titre, description, valeurs clés et tableau alternatif.
