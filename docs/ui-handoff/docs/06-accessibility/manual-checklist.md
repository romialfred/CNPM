# Checklist manuelle d’accessibilité

Pour chaque écran P0 :

- [ ] Navigation complète au clavier, sans piège.
- [ ] Focus visible, non masqué, ordre logique.
- [ ] Zoom 200 % sans perte d’information ni chevauchement.
- [ ] Reflow à 320 CSS px.
- [ ] Lecteur d’écran : titre, landmarks, contrôles et erreurs compréhensibles.
- [ ] Formulaire : labels, obligatoire, aide, erreur et résumé.
- [ ] Contraste texte, icônes, bordures et focus.
- [ ] Statuts compréhensibles sans couleur.
- [ ] Images informatives avec alt pertinent.
- [ ] Tables avec caption et headers.
- [ ] Dialogs/drawers : focus entrant, piégé et restauré.
- [ ] Temps/session : avertissement et récupération.
- [ ] Réduction du mouvement respectée.
- [ ] Mobile : TalkBack/VoiceOver, taille de texte et cibles tactiles.
