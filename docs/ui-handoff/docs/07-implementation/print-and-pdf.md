# Impression et documents

## Impression de pages

Créer des feuilles de style dédiées : masquer navigation/actions, conserver titre, métadonnées, tableaux et pagination logique. Ne pas imprimer un dashboard comme une image.

## Reçu officiel

Le PDF est généré côté service documentaire avec template versionné, signature/cachet officiels selon politique, QR/code de vérification réel et archivage. L’interface affiche un preview ou le PDF rendu mais ne fabrique pas la preuve dans le navigateur.

## Accessibilité

Les PDF destinés au public ou aux membres doivent être balisés selon la cible réglementaire du projet. Fournir aussi les données essentielles dans l’interface HTML.
