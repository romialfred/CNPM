# Internationalisation

## Source

Les chaînes de base se trouvent dans `data/ui-copy.fr.json` et `data/ui-copy.en.json`. Le français est complet ; l’anglais est un squelette à compléter.

## Règles

- Aucune concaténation de fragments traduits.
- Utiliser ICU pour pluriels et genres si nécessaire.
- Les formats de date/nombre viennent de la locale.
- Prévoir une expansion de texte de 30 %.
- Les routes publiques peuvent être localisées ou conserver une structure unique selon décision SEO.
- Les contenus de vitrine sont traduisibles par champ si le CNPM active plusieurs langues.

## Fallback

Si une traduction manque, journaliser en environnement non productif et utiliser le français ; ne jamais afficher la clé brute en production.
