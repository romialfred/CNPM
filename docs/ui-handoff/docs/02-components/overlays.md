# Dialogues, drawers, menus et tooltips

## Dialogue

Utiliser pour une décision courte et bloquante. Largeur 480–640 px, une seule colonne, titre précis, conséquence visible. Le focus entre dans le dialogue, reste piégé, puis revient au déclencheur.

## Drawer

Utiliser pour détails secondaires, filtres sur tablette ou édition légère. Sur desktop : 420–560 px ; sur mobile : pleine largeur ou bottom sheet. Une modification longue devient une page.

## Menu

Les menus contiennent des actions homogènes. Séparer les actions dangereuses et afficher leur libellé complet. Fermer avec Échap et clic extérieur.

## Tooltip

Complément bref, jamais nécessaire pour comprendre un champ ou une action critique. Visible au survol et au focus, refermable avec Échap.
