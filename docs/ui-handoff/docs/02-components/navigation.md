# Navigation

## Back-office

La sidebar étendue mesure 252 px et la topbar 72 px. L’élément actif possède une barre ou un fond bleu très clair, un texte bleu et `aria-current="page"`. La version réduite conserve tooltips et noms accessibles.

## Portail membre

Navigation horizontale sur desktop et bottom navigation sur mobile. Les fonctions essentielles sont : Accueil, Cotisations/Paiements, Reçus, Requêtes, Profil.

## Site public

Header fixe seulement si cela n’occupe pas trop d’espace. Menu mobile dans un drawer. Le CTA « Accéder au portail membre » reste distinct des liens éditoriaux.

## Breadcrumb

Utilisé pour les niveaux profonds du back-office, jamais comme remplacement du titre. La dernière entrée n’est pas un lien.

## Tabs

Les tabs changent une vue locale sans modifier l’objet principal. Si le contenu doit être partageable, synchroniser la tab dans l’URL. Sur mobile, les tabs peuvent défiler horizontalement avec indicateur clair.
