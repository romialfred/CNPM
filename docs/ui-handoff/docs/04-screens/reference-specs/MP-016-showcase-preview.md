# MP-016 — Aperçu et publication de la vitrine

**Route :** `/member/showcase/preview`

L’aperçu rend exactement le template public avec une bannière privée indiquant le statut et la version. Il permet de changer de viewport, vérifier les liens, afficher les données SEO, lancer les contrôles d’accessibilité automatiques et soumettre à modération.

## États

Brouillon non valide, prêt à soumettre, en revue, modifications demandées, approuvé, planifié, publié et dépublié.

## Critères

- [ ] L’aperçu n’est pas indexable et exige authentification.
- [ ] Les liens externes sont marqués et testables.
- [ ] La version affichée correspond à la version soumise.
- [ ] Les erreurs bloquantes sont listées avec liens vers l’éditeur.
