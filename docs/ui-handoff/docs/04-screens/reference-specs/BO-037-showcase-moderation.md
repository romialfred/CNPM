# BO-037 — Modération des vitrines membres

**Route :** `/admin/showcases/moderation`

## Composition

File de demandes à gauche, aperçu/diff au centre, checklist et décision à droite. Le modérateur voit la version publiée et la version proposée, les médias nouveaux, les coordonnées modifiées, les claims commerciaux, le statut d’adhésion et les signalements.

## Actions

Approuver, demander des modifications, rejeter avec motif, planifier, suspendre, dépublier en urgence et restaurer une version approuvée. Toutes les actions sont auditées.

## Critères

- [ ] Motif structuré et commentaire requis pour rejet/suspension.
- [ ] Le modérateur ne modifie pas silencieusement le contenu du membre.
- [ ] Les droits médias sont vérifiables.
- [ ] La décision notifie le membre.
- [ ] La suspension d’un badge et la dépublication de contenu sont des actions séparées.
